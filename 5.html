Application Layer

Information

The Application Layer is the top layer of the OSI model. It allows users to interact with network services through applications such as web browsers, email systems, and file transfer software.

Common Protocols:

  • HTTP – Web communication
  • FTP – File transfer
  • SMTP – Email sending
  • DNS – Domain name translation

Network Layer

Information

The Network Layer is responsible for routing data packets from source to destination across different networks. It ensures that data takes the most efficient path.

This layer manages logical addressing and packet forwarding between networks.

Main Functions:

  • Logical addressing (IP address)
  • Routing data packets
  • Packet forwarding
  • Traffic control
